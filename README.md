# AI Career Path Predictor 💼🔍

🚀 A smart web application that helps students and professionals discover ideal tech career paths by analyzing their resumes and preferences using Natural Language Processing.

### 🔧 Tech Stack
- Python
- Streamlit
- spaCy
- PyMuPDF (for PDF parsing)

### 📄 Features
- Upload and analyze resumes (PDF)
- Ask 5 quick personality questions
- Suggest Top 3 career roles (e.g., ML Engineer, Backend Dev)
- Show personalized learning resources
